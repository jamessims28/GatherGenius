# GatherGenius Layer 1 Marketplace

Project name:

```text
gathergenius-lmh
```

## This safe layer adds

- Marketplace homepage
- Vendor cards
- Search bar
- Category filters
- GatherGenius styling
- Health API route

## Deploy

1. Upload to GitHub repo `gathergenius-lmh`
2. Import or redeploy on Vercel
3. Build command: `npm run build`
4. Node: `20.x`

No Stripe, Supabase, OpenAI, or Maps yet. This keeps the deployment safe.
