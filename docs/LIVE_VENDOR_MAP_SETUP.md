# GatherGenius Live Vendor + Map Setup

Use Vercel project name:

```text
gathergenius-lmh
```

Add both Google keys:

```env
GOOGLE_MAPS_API_KEY=
NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=
```

Run `docs/DATABASE_SCHEMA.sql` in Supabase.

Live features:
- Browser location
- Google map display
- Google Places vendor search
- Vendor application form
- Stripe Connect scaffold
